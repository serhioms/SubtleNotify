Available Endpoints

Endpoint Path Description
actuator /actuator Root endpoint that lists all available actuator endpoints
health /actuator/health Shows application health information
info /actuator/info Displays arbitrary application info
metrics /actuator/metrics Shows metrics information (JVM, CPU, memory, etc.)
metrics/{metric} /actuator/metrics/{requiredMetricName} Shows specific metric details
env /actuator/env Exposes properties from Spring's ConfigurableEnvironment
beans /actuator/beans Displays all Spring beans in the application
configprops /actuator/configprops Shows all @ConfigurationProperties
loggers /actuator/loggers Shows and modifies logger configurations
loggers/{name} /actuator/loggers/{name} Shows/modifies specific logger configuration
httptrace /actuator/httptrace Displays HTTP trace information (last 100 requests)
mappings /actuator/mappings Shows all @RequestMapping paths
scheduledtasks /actuator/scheduledtasks Displays scheduled tasks
threaddump /actuator/threaddump Performs a thread dump
heapdump /actuator/heapdump Returns a heap dump file (binary download)
prometheus /actuator/prometheus Exposes metrics in Prometheus format
caches /actuator/caches Shows available caches
conditions /actuator/conditions Shows auto-configuration conditions
flyway /actuator/flyway Shows Flyway database migrations (if applicable)
liquibase /actuator/liquibase Shows Liquibase database migrations (if applicable)
sessions /actuator/sessions Shows Spring Session information (if applicable)

Configuration

Enable All Actuator Endpoints

Add this to your application.properties:

# Expose all actuator endpoints
management.endpoints.web.exposure.include=*

# Or expose specific endpoints
# management.endpoints.web.exposure.include=health,info,metrics,env,beans

# Exclude specific endpoints
# management.endpoints.web.exposure.exclude=shutdown

# Show detailed health information
management.endpoint.health.show-details=always

# Change actuator base path (optional)
# management.endpoints.web.base-path=/management

# Enable specific endpoints
management.endpoint.health.enabled=true
management.endpoint.info.enabled=true
management.endpoint.metrics.enabled=true

Access URLs

With default configuration (assuming port 8080):

  • Base: http://localhost:8080/actuator
  • Health: http://localhost:8080/actuator/health
  • Info: http://localhost:8080/actuator/info
  • Metrics: http://localhost:8080/actuator/metrics
  • Beans: http://localhost:8080/actuator/beans
  • Mappings: http://localhost:8080/actuator/mappings

Note: Replace localhost:8080 with your actual host and port if different.
